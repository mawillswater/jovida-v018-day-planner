什么样的陪伴 —— 浅互动,深关系
「陪伴」常被画成一条线,在「深度陪伴 / 鼓励式 / 工具」里三选一。但它其实是两个维度:互动时长(每次聊多久)和关系深度(它有多了解你、你有多放不下它)。把这两者分开,答案才清楚。
💕

深度陪伴

每次互动 30–60 分钟
Replika / Tolan / Character.ai
  • best friend / soulmate
  • 长聊天、情感寄托
  • 主要服务孤独感
✕ 长会话和「把事做完」抢时间、也烧 token
🌱

鼓励式陪伴

每次互动 2–5 分钟
Finch / Duolingo / Habitica
  • cheerful sidekick
  • 为小胜利庆祝
  • 高频、低摩擦
✓ 互动时长对了——但只做到这层就太浅
⚙️

轻人格工具

每次互动 5–15 分钟
Sunsama / Notion AI
  • 带一点点人格的工具
  • 优化者心态、偏冷
  • 高效但不暖
✕ 留不住「想变好却坚持不下来」的人
🐰
Jovida 不在这条线上选一个点 —— 而是「浅互动 × 深关系」

互动时长要浅:默认是 2–5 分钟的轻触点,不和「把事做完」抢时间,也不让 token 失控(深聊每天 100 条封顶)。关系深度要深:它记得你说过的话、你坚持了多久、你哪天破防过——一个没有记忆、没有牵挂的「鼓励 sidekick」,只是一个有脸的推送通知,留存撑不过新鲜感。Finch 的高留存,靠的正是用户对那只宠物的深度依恋——深关系不与生产力冲突,它本身就是留存的发动机。Jovida 要的是:像 Finch 一样轻,却比 Finch 更懂你;在你状态很糟的时刻,它也能陪你深聊几句(见 ACT 4 陪伴与情绪)。

04 — 核心机制

三个关键设计

① 软硬分层 —— Agent 怎么生成而不出错

用户最怕 AI 乱排。解法:把一天分两层,Agent 只生成它擅长的那一层。

硬约束
固定的事 —— 只导入,绝不发明、绝不挪动

会议、吃药、健身课、deadline。来自日历 / 提醒事项 / 截图。Agent 永远不碰它 → 「排错一个会议」这种致命失败基本消失。

软推进
朝着目标今天做什么 —— Agent 判断力增值的地方

把「想改善体态」这种模糊愿望,翻译成今天能做、且真的值得做的小事。生成时必须附理由。这才是最难、最有壁垒的活。

② 暖版动力系统 —— 有牵挂,但不羞辱

streak 判定 = 「今天有没有推进」(完成 ≥1 件、或跟 Jovida 说句话),门槛低、很难断。但「门槛低」不等于「没有让人回来的力」—— Finch 靠的正是「宠物会难过」。问题不是要不要这种力,而是这种力指向谁。

✓ 牵挂 —— 力指向「关系」
  • 兔子有真实状态:会想你、好几天没见会担心你
  • 你回来,是不想让这个在乎你的小家伙白等
  • streak / 宝石 / 即时正反馈、角色和你一起长大
✕ 羞辱 —— 力指向「你本人」
  • 红点轰炸、断 streak 的恐吓
  • 「你还有 7 个没做」、进度条没满的焦虑
  • 这些攻击用户本人,把已在自责的人推走
同样是「让你舍不得走」:Finch 的宠物会难过,但从不说「你不行」。我们要那只会牵挂你的兔子,不要那只会指责你的猫头鹰。

③ 抢三个短视频碰不到的时刻

不在「杀时间」上跟抖音 / TikTok 打——赢不了。它们是零启动成本的无限多巴胺机器。Jovida 抢的是它们结构上够不着的时刻。

时刻 01
想变好的那一刻
早上 / 月初 / 刷到励志内容。Agent 已经把计划做好,用户只面对「做一件小事」,不是「规划人生」——拼的是启动成本。
时刻 02
刷完空虚的那一刻
comedown。短视频此刻什么都不给。Jovida 接住,而且绝不骂他。不审判的再进入,是它们给不了的。
时刻 03
被温柔触达的那一刻
它们的推送是纯诱饵。Jovida 踩对时间、语气是温的——一个记得你、在等你的角色,它们结构上没有。

06 — 商业化

钱从哪来,有多大确定性

必须证明:目标用户有明确付费能力,且此场景下有明确付费意愿。

📊
原则 ③

用高 ARR 产品做绝对判断

不自己 YY——拿当下 ARR 能力强的产品作绝对参照。

商业化确定性的 4 层支撑
为什么相信付费率和 ARPU 能成立——从「人群有没有钱」一路推到「我们手上有没有确定的入门盘」。
①
人群已经在付费 —— 钱包是稳的

目标人群每月已经在 ChatGPT、Spotify、流媒体、Finch、Notion 这些订阅上花钱。问题从来不是「付不起」,而是要给一个「值得付」的理由。

②
同类产品已经验证了付费意愿与 ARPU

Finch 7–10% 转化、Tiimo 付费用户 75% 自认神经多样、Sunsama 双位数转化——这条赛道的 unit economics 不用我们从零证明。

③
定价不冒进 —— 和 Finch 同价位,却多给真生产力

$9.99/月、年付 $59.99(≈$5/月),不高于对手;但比 Finch 多了「待办 / 日程 / 专注」这层真生产力,用户更容易理解「为什么该升级」。

④
现有 Jovida 用户 = 高确定性的入门盘

不靠纯冷启动:现有角色付费用户做首月 cross-sell,是早期 ARR 里最稳的一块基本盘。

定价对标 —— Jovida 不靠贵取胜
$9.99/月(年付 ≈$5/月)与 Finch 同价位,远低于 Sunsama / Inflow / Motion。用不高于对手的价格,给「陪伴 + 真生产力」两份价值。

07 — 付费墙

免费把人留住,付费解锁 AI

变现逻辑:免费版 = 纯工具,几乎不烧 token,负责把人留住;付费版 = AI 重的能力,负责变现。onboarding 末尾用「愿景 → 付费墙」强引导,但留一个小入口可跳过;日常里在几个关键时刻用蒙层引导解锁。

免费版 · 负责留住人

纯工具,一直免费

  • 无限 todo / 日历 / 打勾 / streak
  • 一周日历、按时段分组
  • 桌面小组件看今天、基础提醒
  • 每天少量 AI 试用额度
付费版 · 负责变现

解锁完整的 Agent

  • AI 生成 / 重排 / 拆解计划(每天 30 次)
  • 和 Jovida 深聊、情绪陪伴(每天 100 条)
  • 任务拆解 + 配套教程 / 跟练资源
  • 30 天数据趋势、跨天记忆与洞察
  • 全部触达(锁屏 / 灵动岛 / 桌面组件)
